PeriodicTable

A periodic table written in Java using the Swing GUI framework

View on GitHub Download .zip Download .tar.gz

Introduction

This program is a simple way to look up the properties of any element in the periodic table.

Features

This periodic table includes:

  • Ability to lookup elements by atomic number, symbol, or name
  • Name suggestions for invalid searches
  • A tidy, space-saving GUI
  • Ability to display the atomic number, name, symbol, metallic properties, matter state, and atomic mass of each element
  • Use of a file (elements.txt) to store element data
    • Easy extensibility to add elements and element properties in the future ### Downloads The periodic table is built automatically every commit courtesy of drone.io.
